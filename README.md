# Store Rating App

A full-stack web application that allows users to register, browse stores, and submit ratings and reviews. The app calculates average ratings and displays them to help users make informed decisions.  

## Features
- User registration and login with authentication
- Browse stores and view their details
- Submit ratings and reviews for stores
- Display average ratings for each store
- Responsive and user-friendly interface
- CRUD operations for stores and reviews

## Technology Stack
- **Frontend:** React.js, Axios, CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Authentication:** JWT (JSON Web Token)
- **API Testing:** Postman

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/store-rating-app.git
````

2. **Backend setup**

```bash
cd store-rating-backend
npm install
# Configure MySQL database in config file
npm start
```

3. **Frontend setup**

```bash
cd store-rating-frontend
npm install
npm start
```

4. **Access the app**
   Open your browser and go to: `http://localhost:3000`

* `frontend/` – React.js frontend code

* `database/` – MySQL database scripts
